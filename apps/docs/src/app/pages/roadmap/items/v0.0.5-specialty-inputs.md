Five new input primitives that compose on top of `kj-input` / `kj-select`:

- `<kj-cascade-select>` — hierarchical flyout-chain picker.
- `<kj-tree-select>` — flat-rendered hierarchical dropdown with single + multi-select modes.
- `<kj-input-mask>` — fixed-format input with built-in `NG_VALIDATORS`.
- `<kj-input-otp>` — N-cell one-time-code with auto-advance, paste distribution, completion announcement.
- `<kj-input-group>` + `<kj-input-group-addon>` — addon-wrapped fields with `aria-labelledby` composition.
