Image in, theme out. Run a k-means / median-cut palette extraction, map clusters onto the kj token slots (primary / accent / surface / fg / bg), score the result against WCAG, then hand the user a forked theme they can refine in the theme generator. Cuts the cold-start barrier for new adopters.
