`KjTableDirective` wraps `@tanstack/angular-table` so consumers get TanStack's row model, sort/filter/group plumbing, and column virtualization for free. The styled wrapper adds the WAI-ARIA Grid pattern on top: `role="grid"` with proper row/cell roles, `aria-rowindex` / `aria-colindex` for virtualized rows, `aria-sort` reflection on sortable headers, full keyboard nav (Arrow / Home / End / PageUp-Down, type-ahead), and a live-region announcement when sort or filter mutates the row set. Resize handles persist widths to a configurable store (default: in-memory; opt-in localStorage).
