`@kouji-ui/core` already ships a basic `kj-table` primitive. The styled wrapper adds the production pieces: virtual scrolling (`cdk-virtual-scroll-viewport`), per-column sort/filter, multi-sort with shift-click, resize handles with persisted widths, server-driven loading mode, and a pagination subcomponent.
