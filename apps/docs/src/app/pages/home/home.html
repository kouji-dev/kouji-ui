Why kouji
◈

Directive-first

Everything in core is a directive. Compose behaviors via hostDirectives. No components, no templates, no opinions.

⚡

Signals everywhere

Built for Angular 21. Signal inputs, signal contexts, afterNextRender. No RxJS, no lifecycle interfaces.

♿

WCAG 2.1 AAA

Every component targets AAA compliance. Full keyboard nav, ARIA patterns, focus management, live regions.

◎

CDK-powered

Wraps Angular CDK primitives for overlays, menus, listbox, accordion. Battle-tested patterns, not reinvented.

∅

Zero-CSS core

The core has no styles. Bring Tailwind, CSS modules, vanilla CSS, or anything else. The impl package ships Tailwind v4.

⊞

TanStack + ECharts

Complex data components powered by TanStack Table for data grids and Apache ECharts for charts.

Start in
