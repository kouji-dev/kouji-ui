# kouji-ui — Phase 1: Monorepo Bootstrap

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scaffold the kouji-ui monorepo — Angular 21 workspace, `@kouji-ui/core` library, `@kouji-ui/ui` library, `apps/docs` SSR application, Turborepo pipeline, Vitest, Playwright, Changesets, and GitHub Actions CI. No component code — only tooling and scaffolding.

**Architecture:** Single Angular workspace (`angular.json`) manages two libraries and one SSR app. pnpm workspaces handles cross-package resolution and publishing. Turborepo drives build ordering and caching. Consecutive execution throughout — each step depends on the previous.

**Tech Stack:** Angular CLI 21, pnpm 9+, Turborepo 2+, @angular/cdk ^21, @tanstack/table-core, echarts, tailwindcss ^4, @analogjs/vitest-angular, vitest, @playwright/test, @changesets/cli

**This is Plan 1 of 4.**
- Plan 2: Core Shared Primitives (consecutive — base directives other components depend on)
- Plan 3: Core Component Directives (parallel agents per directive)
- Plan 4: UI Package + Docs App (parallel agents per component/page)

## Task 13: Set Up GitHub Actions CI

**Files:**
- Create: `.github/workflows/ci.yml`
- Create: `.github/workflows/release.yml`

- [ ] **Step 1: Create .github/workflows directory**

```bash
mkdir -p .github/workflows
```

- [ ] **Step 2: Create CI workflow**

Create `.github/workflows/ci.yml`:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  ci:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: pnpm/action-setup@v4
        with:
          version: 9

      - uses: actions/setup-node@v4
        with:
          node-version: 22
          cache: pnpm

      - name: Install dependencies
        run: pnpm install --frozen-lockfile

      - name: Install Playwright browsers
        run: pnpm exec playwright install --with-deps chromium

      - name: Lint
        run: pnpm lint

      - name: Build
        run: pnpm build

      - name: Test
        run: pnpm test

      - name: E2E
        run: pnpm test:e2e
```

- [ ] **Step 3: Create release workflow**

Create `.github/workflows/release.yml`:

```yaml
name: Release

on:
  push:
    branches: [main]

concurrency: ${{ github.workflow }}-${{ github.ref }}

jobs:
  release:
    runs-on: ubuntu-latest
    if: github.repository == 'YOUR_ORG/kouji'

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: pnpm/action-setup@v4
        with:
          version: 9

      - uses: actions/setup-node@v4
        with:
          node-version: 22
          cache: pnpm
          registry-url: 'https://registry.npmjs.org'

      - name: Install dependencies
        run: pnpm install --frozen-lockfile

      - name: Create Release PR or Publish
        uses: changesets/action@v1
        with:
          publish: pnpm release
          version: pnpm version-packages
          commit: 'chore: version packages'
          title: 'chore: version packages'
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          NODE_AUTH_TOKEN: ${{ secrets.NPM_TOKEN }}
```

> **Note:** Replace `YOUR_ORG/kouji` with the actual GitHub repository path before publishing.
